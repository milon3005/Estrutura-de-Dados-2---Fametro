
# Questão 2 v2 - Fila Ingressos

## Compilação e Execução
```bash
gcc main.c fila.c -o ingressos2
./ingressos2
```

## Exemplo de Uso
```
Ingressos: 10
ADD C1 4
ADD C2 8
START
REL
EXIT
```

## Complexidade (Big-O)
- Enfileirar: O(1)
- Desenfileirar: O(1)
- Verificar se Fila está Vazia: O(1)
- Limpar Fila: O(n)
